# SQLAlchemy


## SQLAlchemy 란?
- Python에서 사용 가능한 ORM(Object-relational mapping)
    - ORM이란 객체(Object)와 관계(Relation)를 연결해주는 것
    - Object 필드 <-- 매핑 --> Database 데이터


### 장점
- 객체 지향적인 코드로 비즈니스 로직에 집중 가능
- 재사용 및 유지보수 편리성 증가
- DBMS에 대한 종속성이 줄어듬


### 단점
- ORM 만으로 서비스를 구현하기 어려움
- 프로시저가 많은 시스템에서는 장점을 가져가기 어려움
