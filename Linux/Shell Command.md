# 쉘 명령어(Shell Command)
&nbsp;
## 1. `ls`
> 파일 목록을 확인하는 명령어

&nbsp;
## 2. `cd`
> 디렉토리 위치를 변경하는 명령어

&nbsp;
## 3. `mkdir`
> 디렉토리를 생성하는 명령어

&nbsp;
## 4. `mv`
> 파일이나 디렉토리를 이동하는 명령어

&nbsp;
## 5. `rm`
> 파일을 삭제하는 명령어

&nbsp;
## 6. `rmdir`
> 디렉토리를 삭제하는 명령어

&nbsp;
## 7. `chmod`
> 파일이나 디렉토리의 권한을 변경하는 명령어

&nbsp;
## 8. `pwd`
> 현재 디렉토리를 확인하는 명령어

&nbsp;
## 9. `grep`
> 입력으로 전달된 파일의 내용에서 특정 문자열을 찾고자할 때 사용하는 명령어

&nbsp;
## 10. `ps`
> 현재 진행 중인 프로세스를 확인할 수 있는 명령어

&nbsp;
## 11. `df`
> 디스크 전체 남은 용량을 확인하는 명령어

&nbsp;
